Álgebra

Conceito

Álgebra é a parte da matemática que estuda relações, equações e expressões envolvendo incógnitas. É fundamental para resolver problemas de diversas áreas.

Fórmulas importantes

  • Equação do 1º grau: ax + b = c → x = (c-b)/a
  • Produtos Notáveis: (a+b)² = a² + 2ab + b², (a-b)² = a² - 2ab + b²
  • Fatoração: a² - b² = (a-b)(a+b)

Exemplo: Resolver 3x + 7 = 19

Resolução: 3x = 19 - 7 → 3x = 12 → x = 4

Dica: Sempre isole a variável passo a passo, verificando cada operação.

Questão 1: Resolva a equação 2x + 5 = 15

Comentário: Subtraia 5 dos dois lados: 2x = 10 → x = 5.

Como estudar: Pratique equações do 1º grau isolando a variável, comece com valores pequenos e vá aumentando a dificuldade.

Geometria

Conceito

Estuda formas, figuras, medidas, ângulos, áreas e volumes. Fundamental para visualizar e calcular dimensões de objetos.

Fórmulas importantes

  • Área do círculo: A = πr²
  • Perímetro do quadrado: P = 4 × lado
  • Volume do cubo: V = lado³

Exemplo: Área de um círculo de raio 7cm

Resolução: A = π × 7² ≈ 3,14 × 49 ≈ 153,86 cm²

Dica: Memorize a fórmula e pratique com diferentes raios.

Questão 1: Qual é a área de um círculo com raio 7 cm?

Comentário: A ≈ π × 7² = 153,86 cm² → alternativa correta A.

Como estudar: Faça exercícios de áreas de círculos, triângulos e quadrados para fixar as fórmulas.

Funções

Conceito

Funções relacionam valores de entrada (x) a saídas (y). É importante saber interpretar gráficos e expressões algébricas.

Exemplo: f(x) = 2x + 3. Para x = 4 → f(4) = 2*4 + 3 = 11

Dica: Pratique valores diferentes para x e veja como y muda.

Probabilidade e Estatística

Conceito

Estude média, mediana, moda, gráficos, tabelas e probabilidade básica. Fundamental para interpretar dados.

Exemplo: Média de 4, 6, 8 → (4+6+8)/3 = 6

Raciocínio Lógico

Conceito

Resolver problemas usando lógica, relações e sequências. Importante para interpretar enunciados e deduzir respostas.

Exemplo: Se todos A são B e C é A, então C é B.
